04 — The Knowledge Layer

Persistent AI memory, built into the plugin.

Not an external service. Not a config file. Database tables inside WordPress — kl_* — that the AI reads and writes every session. It arrives knowing the site's history.

The behavioral directive every AI receives on connection:

mode: inquiry
allowed: [ "read" ]
restricted: [ "write", "delete" ]
lifts: when the user gives a directive

The AI reads everything. Writes nothing. Waits.
